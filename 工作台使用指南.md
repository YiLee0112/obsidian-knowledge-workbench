# 工作台使用指南

1. 新输入先放进 [[Inbox/快速捕获|快速捕获]] 或标记为 `status: inbox`。
2. 值得保留但还不成熟的内容进入 `seed`。
3. 正在补充证据和链接的内容进入 `growing`。
4. 能独立复用的观点升级为 `evergreen`。
5. 不再活跃但仍需保留的内容标记为 `archive`。

工作台的“本周入库”和热力图读取 `updated ?? created ?? date`。
